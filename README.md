0. virture env 구성

```
python -m venv ./env
```

1. virture enviroment start

```
source env/bin/activate
```

2. virture env안에 lib 설치

```
pip install -r requirements.txt
```

3. start py (아래 명령어로 실행 안되면 env로 시작하는 명령어 실행)

```
python main.py
```

```
env python main.py
```

99. virture enviroment quick

```
deactivate
```
